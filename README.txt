What is this?
-------------

This is MZ-IME Japanese Input (mzimeja).
This program is a Japanese Input Method Editor (IME) for Windows.

Hotkey Support
--------------

For non-Japanese keyboards (e.g., US, German layouts), you can toggle 
the IME ON/OFF using Alt+~ (Alt+VK_OEM_3).

For more details, please refer to READMEJP.txt (Japanese documentation).
