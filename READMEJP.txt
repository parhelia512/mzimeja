「MZ-IME日本語入力」by 片山博文MZ

これは何？
----------

これは、片山博文MZによる「MZ-IME日本語入力」(mzimeja)です。
このプログラムは、Windows で動作する日本語入力システム（IME）です。

MZ-IMEの特徴
------------

 * 勝手にインターネットに接続しません。
 * キー入力や変換結果を記録しません。学習しません。
 * MIT ライセンスでオープンソースです。
 * ユーザー辞書に単語を登録できます。
 * 人名・地名辞書搭載。郵便番号変換が可能です。

動作環境
--------

 * Windows 2000 / XP / Server 2003 / Vista / 7 / 10 / 11 / ReactOS

Windows 2000にインストールしたいときは、低レベル セットアップ (mzimeja-...-LL-setup) をお使いください。

使い方
------

操作方法は、普通のIMEとほとんど同じです。
ただ、かな漢字変換があまり賢くないので、変換がうまくいかないときは、
文節の長さを[Shift]+[←]と[Shift]+[→]で調節したり、何度も変換を繰り返す必要があります。

    表1. キーボードの操作方法
    -------------------------------------------------
    (キー入力)              (説明)
    -------------------------------------------------
    [半角／全角]            IMEのON/OFFを切り替え
    [スペース]              かな漢字変換
    [変換]                  かな漢字変換
    [Esc]/[Escape]          元に戻す／キャンセルする
    [BkSp]/[BackSpace]      直前の文字を消す
    [Del]/[Delete]          直後の文字を消す
    [Enter]                 決定
    [かな]                  ひらがな入力に切り替える
    [Shift]+[かな]          カタカナ入力に切り替える
    [Alt]+[かな]            ローマ字入力を切り替える
    [CapsLock]              英数字入力に切り替える
    [Shift]+[CapsLock]      大文字小文字の切り替え
    [F5]                    コード入力
    [F6]                    全角ひらがなに変換
    [F7]                    全角カタカナに変換
    [F8]                    半角カタカナに変換
    [F9]                    全角英数字に変換
    [F10]                   半角英数字に変換
    [無変換]                漢字に変換しない
    [↑]                    上へ移動する
    [↓]                    下へ移動する
    [←]                    左へ移動する
    [→]                    右へ移動する
    [Shift]+[←]            文節を短くする
    [Shift]+[→]            文節を長くする
    -------------------------------------------------

漢字の入力を助けるために「IMEパッド」というプログラムがあります。
IMEパッドを使えば、総字画や部首から漢字を選んで入力できます。

注意事項
--------

 * 辞書にない単語は、変換ができないことがあります。
